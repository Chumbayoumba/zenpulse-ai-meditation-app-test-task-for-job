2. Что тебе нужно сделать руками (я это за тебя сделать не могу)
---------------------------------------------------------------
1) Если на Android Expo Go пишет, что версия не подходит:
   - скачай актуальный APK отсюда:
     https://expo.dev/go
   - Play Store версия иногда отстает от Expo SDK 55

2) Запусти проект:
   Терминал #1:
   npm run ai-proxy

   Терминал #2:
   npx expo start --lan --clear

3) Открой Expo Go на телефоне и отсканируй QR-код

4) Если bundle открывается, но AI-generation не работает по сети:
   - создай/обнови `.env.local`:
     EXPO_PUBLIC_AI_ENDPOINT=http://<LAN_IP_ТВОЕГО_ПК>:8788/api/affirmation
   - потом перезапусти Expo

5) Запиши обязательный screencast на 7-12 минут

6) При необходимости отдельно запиши короткое demo-video с телефона:
   - Paywall
   - limited access
   - locked redirect
   - premium unlock
   - session flow
   - AI affirmation

5. Готовые примеры prompt'ов для screencast
-------------------------------------------
Prompt 1 — дизайн paywall:
Create a premium meditation paywall in React Native + Expo.
The visual style should feel luxury wellness, editorial, soft pink / plum / lavender,
with expensive-looking spacing, strong hierarchy, Safe Area support, and mobile-first layout.
Include monthly and yearly plans, highlight the better-value option, and add a strong
"Try premium free" CTA. Avoid generic AI-slap card layouts and avoid unreadable text on gradients.

Prompt 2 — фикс мобильной верстки:
The paywall layout must stay readable on iPhone SE and large phones.
Please fix any overflow, cramped spacing, or CTA collisions.
Keep all critical actions reachable, respect Safe Area, and preserve tap-target sizes.

Prompt 3 — locked logic:
If isSubscribed = false, premium meditation cards must look locked and remain tappable,
but tapping them should route back to the paywall instead of opening premium content.
